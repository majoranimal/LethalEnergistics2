// Storage Visuals:
// ---- Item import/export chamber: A circular chamber with glass walls (think test-tube) and a transparent door, there is an 
//                                  iris-door above the chamber and a hinged circular section as the floor of the chamber
//
// ---- Storage compartment: Part of the ships floor becomes transparent and the bottom of the ship is expanded downward, the
//                           external walls remain opaque with a chute at one side (think Ultrakill doors)
//
// ---- Importing: The hinged section falls down (think trapdoor) and the contents of the chamber are dropped into a pitch-black
//                 area below the chamber, the players hear a machine rumbling before the items are shot out at a random angle
//                 into the storage compartment through the chute, the items remain in the same position until exported
//
// ---- Exporting: The iris door opens and you see a "wind sucking" effect between the chute and chosen items, the delay between
//                 each item is 1.5 seconds to prevent them from intersecting

// Player interactions:
// ---- Standing in the chamber during import: Player is dropped into the darkness, they hear a machine-rumbling and get 
//                                             constant fear, the machine stops and they're sucked into an under-floor
//                                             storage compartment forced to look up at the other players, the player is
//                                             a cube (think wall-e scrap cubes) to the other players and upon being exported
//                                             they remain as a cube, the cube weighs 4 lb and can be thrown like an egg with
//                                             a larger parabola, when scanned it shows "Bodily damage caused by: Compaction" 
//
// ---- Standing in the chamber during export: A chute above the player opens and items are rapidly dropped dealing low damage
//                                             with each hit, if enough items are dropped the player dies with cause of death
//                                             being "blunt object trauma"
